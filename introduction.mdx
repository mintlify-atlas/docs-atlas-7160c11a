## What is HTTPToRequests?

HTTPToRequests is a powerful tool that automatically converts HTTP messages into clean, production-ready Python requests code. Say goodbye to manually translating HTTP requests or doing curl-to-requests conversions.

Whether you're a security researcher working with Burp Suite, a developer testing APIs, or an automation engineer, HTTPToRequests saves you time by generating PEP8-compliant Python code from HTTP message files.

## Example Output

From a Burp Suite export, HTTPToRequests generates clean code like this:

```python
import requests

def post_api_login(session):
    headers = {'Host': 'example.com', 'Content-Type': 'application/json', 'Authorization': 'Bearer example_token'}
    json = {
        "username": "test_user",
        "password": "secure_password"
    }
    
    response = session.post('https://example.com/api/login', headers=headers, json=json)
```
